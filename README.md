## Installation
Spooky is developed using Flutter 3.0.0. If you have a different version on your machine, please consider [installing FVM](https://soksereyphon8.medium.com/flutter-version-management-3c318c4ff97d). 

```s
fvm flutter pub get
fvm flutter run
```
